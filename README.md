# Hypercare React Take Home Assignment

Create a client-side Todo app using React. This is expected to be completed in under 2 hours.

Required features:

1. Display todos loaded from `assets/todos.ts`.
2. User can add a todo item
3. User can mark a todo item as complete, or incomplete again
4. User can delete a todo item.
5. User is prompted with a confirmation dialog before an item is deleted.

Bonus 1: Error handling.
Bonus 2: Unit tests for components or business logic.
Bonus 3: use a React-based server-rendered framework to ensure the user experience does not depend on browser-side JS.

Out of scope:

- Persisted data handling.

Grading rubric:

- Accessibility: 50%
- Code organization: 25%
- Component abstractions: 15%
- Style and layout: 10%
- Bonuses or creativity earn extra points, but you should try to complete the assignment in under 2 hours.
- AI use is allowed, but you'll need to be able to fully explain your solution.

To get started, fork this example project and save your work; or use a React-based server-rendered framework.

Once completed, please email a link to your project to <enrico.sacchetti@hypercare.com>.
